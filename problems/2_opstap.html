In deze opdracht is het vooral de bedoeling om te bepalen wat de output is van de code. Test je antwoorden met behulp van de python tutor.

print("staat buiten de if statement")

print("staat buiten de if statement")

L = ['pi', "isn't", [4,2]]

  1. Wat is len(pi)?

  2. Wat is len(L)?

  3. Wat is len(L[1])?

  4. Wat is pi[2:4]?

  5. Welk puntje (‚Äúslice‚Äù) pi is gelijk aan [3,1,4]? (hint, neem het aantal elementen als stop waarde)

  6. Welk puntje pi is [3,4,5]?

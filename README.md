# PGM online boek

Deze repository wordt gebruikt om het online boek voor PGM1 en PGM2 te bouwen. De up-to-date build van het boek [kan hier gevonden worden](https://hanze-hbo-ict.github.io/programmeren/about/syllabus.html).


## Aanpassingen

Hoewel we erg ons best doen het boek up-to-date te houden, kan het soms toch gebeuren dat er een foutje in sluipt. Spot je zo'n foutje? Help ons dan vooral! Je kan problemen, suggesties, etc melden via het [anonieme feedback formulier](https://hanze-hbo-ict.github.io/programmeren/about/feedback.html), maar natuurlijk is het ook een optie deze repo te forken, de fout te herstellen, en een PR te openen!


## Bronnen

- Dit boek is gebaseerd op [deze template repository](https://github.com/firasm/jupyterbook_course_template).
- De inhoud van het vak steunt op het boek [Think Python. How to Think Like a Computer Scientist van Allen Downey](https://greenteapress.com/wp/think-python-3rd-edition/)
